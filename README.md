# golang-tasks

## Getting started 

```
$ cd $HOME
$ mkdir code
$ cd code
$ git clone https://github.com/zacksfF/golang-tasks.git
$ cd golang-tasks
```

### Challenge 1: Data Fragmentation and Reconstruction

```
cd Data_Fragmentation_and_Reconstruction
go run main.go 

#Output
Reconstructed Data: HelloGolang!
```
